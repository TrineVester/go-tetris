# Go Tetris (Terminal)

A simple Tetris clone you can play directly in a terminal window.

## Requirements

- Go 1.22+
- A terminal that supports ANSI escape sequences (Windows Terminal, PowerShell, or VS Code terminal)

## Run

```bash
go run .
```

## Controls

- Left/Right: move
- Down: soft drop
- Up: rotate
- Space: hard drop
- Q: quit

## Notes

If you see no colors or the screen does not update, try running in Windows Terminal.
